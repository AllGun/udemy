Espaçamento de letras

A propriedade letter-spacing no CSS é usada para controlar o espaço entre caracteres em um elemento de texto.

Valores possíveis:

  • Normal: é o valor padrão
    .class-name {
    letter-spacing: normal;
    }
  • Número positivo: Aumenta o espaço entre os caracteres.
    .class-name {
    letter-spacing: 1px;
    }
    .class-name {
    letter-spacing: 0.5em;
    }
    .class-name {
    letter-spacing: 2rem;
    }
  • Número negativo: Diminui o espaço entre os caracteres.
    .class-name {
    letter-spacing: -1px;
    }
    .class-name {
    letter-spacing: -0.5em;
    }
    .class-name {
    letter-spacing: -2rem;
    }
  • Length values: Você pode usar unidades como pixels (px) para definir um valor específico.
