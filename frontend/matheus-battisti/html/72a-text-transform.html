A propriedade text-transform no CSS é utilizada para controlar a transformação do texto, alterando a capitalização das letras. Aqui estão os valores que você pode utilizar:

  • none: Nenhuma transformação. O texto é exibido como foi inserido.
  • capitalize: Transforma o primeiro caractere de cada palavra para maiúsculo.
  • uppercase: Transforma todos os caracteres para maiúsculo.
  • lowercase: Transforma todos os caracteres para minúsculo.
